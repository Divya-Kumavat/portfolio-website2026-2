# Portfolio Website Deployment using Docker and AWS EC2

## Project Overview

This project demonstrates the deployment of a static portfolio website using Docker on an AWS EC2 instance.

## Technologies Used

* HTML
* CSS
* Docker
* AWS EC2
* GitHub

## Deployment Steps

1. Developed a static portfolio website using HTML and CSS.
2. Created a Dockerfile using Nginx as the base image.
3. Built a Docker image locally.
4. Launched an AWS EC2 instance.
5. Installed Docker on EC2.
6. Cloned the GitHub repository on EC2.
7. Built and ran the Docker container.
8. Configured Security Groups to allow web traffic.
9. Accessed the website using the EC2 Public IP.
